2  数据预处理

本节主要对表数据（data.frame）进行详解，包括增、删、查、改等单表操作，以及表之间数据关联的操作，并包含一些向量化计算的内容。 此外还会涉及到空间数据中坐标系的转换问题。

为了方便，称 data.frame 类型的数据，统称为“表数据”，这里“表”指数据类型，而不是文件格式。

2.1 单个表数据的操作

上节已经讲到，用列表示的一个或更多的数据种类，每行包含一个唯一的数据实体。 因此，对表数据的操作将从行、列两个角度进行展开。 我们将单个表数据的操作归为“创建、增加、查询、修改、删除”五个类型。

2.1.1 创建

创建一个表，是使用 data.frame() 函数进行的。创建表时，需要指定这个表里面有哪些列，每一列有哪些数据。例如我们使用如下语句创建一个表

demo.suspect <- data.frame(
    name = c("张三", "李四", "王五"),
    age = c(20, 26, 30),
    gender = c("M", "F", "M"),
    height = c(1.78, 1.65, 1.70),
    weight = c(80, 45, 60),
    shoe.size = c(43,38,41)
)
demo.suspect
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43
2 李四  26      F   1.65     45        38
3 王五  30      M   1.70     60        41

函数 data.frame() 具有如下参数：

参数名 类型 含义 默认值
... 变长参数 每一列的值。可以指定参数名，也可以不指定。如果指定参数名，则使用参数名作为列名。
row.names integer/character/vector(integer/character) 指定行名 NULL
check.rows logical 检查列名是否符合语法、是否存在重复等问题 FALSE

需要指出的是：

  • 每列的值不一定要是向量；如果是向量，则各列值一定要一样长。
  • 可以不指定任何列，以创建一个空表。但建议提前创建好表结构，以提高程序运行速度。
data.frame(
    name = c("张三", "李四", "王五"),
    age = c(20, 26, 3),
    gender = "M"
)
  name age gender
1 张三  20      M
2 李四  26      M
3 王五   3      M
data.frame()  # 空表数据
data frame with 0 columns and 0 rows

2.1.2 查询

查询一行或者一列，可以使用与查看矩阵相同的方法，即使用 [] 运算符。 可以像矩阵一样，在 [] 中使用整数、序列、整型向量、逻辑型向量或者省略行列索引其中之一来获取数据。

demo.suspect[c(1,2), c(2,3)]
  age gender
1  20      M
2  26      F

对于具有列名或行名的矩阵或表数据，可以通过指定其行名、列名来获取数据

demo.suspect[, c("name", "height", "weight")]
  name height weight
1 张三   1.78     80
2 李四   1.65     45
3 王五   1.70     60

如果只取某一列的所有值，可以使用运算符 $ 获取这一列的值，例如

demo.suspect$gender
[1] "M" "F" "M"

还可以使用运算符 [[ ]]，中间既可以填写列索引，也可以填写列名，例如

demo.suspect[["gender"]]
[1] "M" "F" "M"
demo.suspect[[3]]
[1] "M" "F" "M"

以上只是对数据简单的查找。能否像数据库一样，给数据中指定一些条件，筛选出符合条件的数据呢？答案是可以的，而且有多种方式可以做到。这里介绍三种。

第一种是直接使用逻辑判断表达式，得到一组逻辑类型的向量，根据该向量获取对应位置上的元素。例如提取 demo.suspect 中 gender 列为 M 的行，可以这样操作

demo.suspect[demo.suspect$gender == 'M',]
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43
3 王五  30      M   1.70     60        41

第二种是稍微有些多此一举，使用 which() 函数。这个函数接收一个逻辑类型的向量作为参数（如果不是向量，也会被转换成向量），返回这些向量中值为 TRUE 的元素的位置。例如

which(demo.suspect$gender == 'M')
[1] 1 3
demo.suspect[which(demo.suspect$gender == 'M'),]
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43
3 王五  30      M   1.70     60        41

第三种是使用一个强大的 dplyr 包，该包提供了 select 和 filter 两个函数，可以大大简化对数据的查找。 其中，select 函数偏向于获取列，filter 函数偏向于获取行，基本对应了 SQL 语句中 select 和 where 两个子句，但提供了更多的功能。

以下是一些 select 函数的示例

library(dplyr)

载入程辑包：'dplyr'
The following objects are masked from 'package:stats':

    filter, lag
The following objects are masked from 'package:base':

    intersect, setdiff, setequal, union
select(demo.suspect, name)  # 获取 name 列
  name
1 张三
2 李四
3 王五
select(demo.suspect, c("height", "weight", "shoe.size"))
  height weight shoe.size
1   1.78     80        43
2   1.65     45        38
3   1.70     60        41
select(demo.suspect, height:shoe.size)  # 获取 height 到 shoe.size 等列
  height weight shoe.size
1   1.78     80        43
2   1.65     45        38
3   1.70     60        41
select(demo.suspect, ends_with("ght"))  # 获取以 "ght" 结尾的列
  height weight
1   1.78     80
2   1.65     45
3   1.70     60

starts_with 和 ends_with 是 dplyr 提供的判断字符串开头和结尾的两个函数，

可以看到，该函数可以接收多种类型的参数：字符串、字符串向量、:表示的序列，以及一些函数返回值。

以下是一些 filter 函数的使用示例

filter(demo.suspect, name == "张三")
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43
filter(demo.suspect, gender == "M" & age > 20)
  name age gender height weight shoe.size
1 王五  30      M    1.7     60        41
filter(demo.suspect, gender == "M", age > 20)
  name age gender height weight shoe.size
1 王五  30      M    1.7     60        41
filter(demo.suspect, between(age, 20, 30))
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43
2 李四  26      F   1.65     45        38
3 王五  30      M   1.70     60        41
filter(demo.suspect, gender == "M", ((height * 100 - 80) * 0.7) < weight)
  name age gender height weight shoe.size
1 张三  20      M   1.78     80        43

可见，该函数还是主要接收一个逻辑类型的向量作为筛选依据，可以看作是对第一种数据检索方式的简化。 该函数也可以接收多个条件，多个条件之间是使用 & 运算符连接的。 该函数还可以在列之间进行比较，也可以对列做一些运算后比较。 该函数还支持一些函数，用于方便进行数据检索：

  • between(x, L, U) 判断变量 \(x\) 的元素是否落在 \([L,U]\) 的区间内
  • near(x, c) 对于浮点数 \(x\) 来说，直接使用 x == c 来判断 \(x\) 和 \(c\) 是否相等，会存在一些问题。该函数是用来判断 \(x\) 和 \(c\) 是否在浮点误差内相等。
  • is.na(x) 判断变量 \(x\) 是否是 NA

当然这两个函数可以进行嵌套

select(filter(demo.suspect, shoe.size < 42), "name":"gender")
  name age gender
1 李四  26      F
2 王五  30      M

2.1.3 分组查询

在 R 语言中，还可以对数据进行分组查询，相当于 SQL 中的 group by 子句。实现这一操作有几种方式。

一种是使用 dplyr 包中提供的 group_by() 函数，该函数给数据指定一个分组依据，但不对数据进行分组处理。使用 summarise() 或 summarize() 函数进行分组处理。 使用一次 group_by() 函数，数据中增加一层分组条件；使用一次 summarise() 函数，数据中减少一次分组条件。

summarise(group_by(demo.suspect, gender), height.mean = mean(height), weight.mean = mean(weight))
# A tibble: 2 × 3
  gender height.mean weight.mean
  <chr>        <dbl>       <dbl>
1 F             1.65          45
2 M             1.74          70

这里用到了 mean() 和 length() 两个函数，其作用分别是计算平均数和向量长度。

这种方式就非常类似于 SQL 语句的方式，使用起来也非常方便。

另一种是使用 R 自带的 aggregate 函数，对数据进行聚合。例如

aggregate(demo.suspect[, c("height", "weight")], by = list(demo.suspect$gender), FUN = mean)
  Group.1 height weight
1       F   1.65     45
2       M   1.74     70

该函数具有如下参数：

参数名 类型 含义 默认值
x data.frame 要处理的数据
by list 分组依据
FUN function 聚合函数。将每一列传入到函数的第一个参数中。
... logical 聚合函数的额外参数

这里涉及到两个类型 list 和 function ，即列表类型和函数类型。

2.1.3.1 列表类型

列表类型，有点类似于其他编程语言的字典类型（C++ 中对应 Map 类型），是以键值对方式存在的数据。 其中值必须有，但键名不一定必须有。当键名有时，可以比较类似于字典；当键名没有时，比较类似于任意列表。 因此该类型的使用非常灵活，可以和 JSON 数据对应。

创建列表类型变量的方式和创建表数据的方式很像，使用如下方式

list(
    name = "张三", gender = "Male", age = 30, 
    children = list(
        list(name = "张军", gender = "Male", age = 3),
        list(name = "张灵", gender = "Female", age = 2)
    ),
    interests = c("体育", "音乐")
)
$name
[1] "张三"

$gender
[1] "Male"

$age
[1] 30

$children
$children[[1]]
$children[[1]]$name
[1] "张军"

$children[[1]]$gender
[1] "Male"

$children[[1]]$age
[1] 3


$children[[2]]
$children[[2]]$name
[1] "张灵"

$children[[2]]$gender
[1] "Female"

$children[[2]]$age
[1] 2



$interests
[1] "体育" "音乐"

事实上，data.frame 类型只是一个特殊的 list 类型，特殊在不同键的值具有相同的长度。

在 aggregrate 函数中，如果 by 参数传入的列表只有几个键值对，那么就按照这几个条件进行分组。多个条件分组的示例如下

aggregate(demo.suspect[, c("height", "weight")], by = list(demo.suspect$gender), FUN = mean)
  Group.1 height weight
1       F   1.65     45
2       M   1.74     70

2.1.3.2 函数类型

R 中，函数也是一个变量，即函数类型的变量（有点 JavaScript 那味了），但是 () 运算符只能作用于函数类型的变量上。

创建函数类型的变量使用 function 关键字，将最后一条运行的语句的结果作为返回值。例如，创建一个计算二次函数值的变量

quadratic <- function(x, a = 1, b = 0, c = 0) {
    (a * x^2 + b * x + c)
}
quadratic(1)
[1] 1
quadratic(1, 1, 2)
[1] 3
quadratic(1, 1, 2, 1)
[1] 4

R 语言中的函数无需使用 return 指明返回值， return() 也是一个函数，但其只起到控制函数流程的作用。

R 语言中也提供了丰富的函数，用于进行简单的统计计算，包括 mean(), sd(), var(), min(), max(), median(), length(), range(), quantile(), fivenum() 等。 这些函数可以直接使用。

mean(1:5)       # 平均数
[1] 3
sd(1:5)         # 标准差
[1] 1.581139
var(1:5)        # 方差
[1] 2.5
min(1:5)        # 最小值
[1] 1
max(1:5)        # 最大值
[1] 5
median(1:5)     # 中位数
[1] 3
length(1:5)     # 长度
[1] 5
range(1:5)      # 极值
[1] 1 5
quantile(1:10)  # 四分位数
   0%   25%   50%   75%  100% 
 1.00  3.25  5.50  7.75 10.00 
fivenum(1:10)   # 四分位数（与上一种算法不同）
[1]  1.0  3.0  5.5  8.0 10.0

2.1.3.3 判断语句

在之前可以看到，我们调用 mean 作为统计函数的时候，如果遇到字符串或逻辑值，其统计结果是有问题的。这时就需要我们使用自定义的函数，对不同类类型的列进行处理

aggregate(demo.suspect, by = list(demo.suspect$gender), FUN = function(x) {
    if (class(x) == "character") {
        length(x)
    } else if (class(x) == "logical") {
        length(which(x))
    } else {
        mean(x)
    }
})
  Group.1 name age gender height weight shoe.size
1       F    1  26      1   1.65     45        38
2       M    2  25      2   1.74     70        42

这里使用到了 if else 关键字进行函数流程的控制。 if 关键字后面使用 () 包含一个逻辑表达式表示条件，当该条件成立时执行 {} 中的语句；不成立时，如果有 else 则执行 else 后 {} 的语句。

2.1.3.4 循环语句

。 与其它语言类似，R 语言中也有 for while 两种循环语句。这里顺便介绍一下。

for 语句的用法是

for (i in demo.suspect$height) {
    print(i * 100)
}
[1] 178
[1] 165
[1] 170

这里的 print() 函数将变量进行输出。

while 语句的用法是

i <- 2
while (i > 0.0001) {
    i <- i / 2
}
i
[1] 6.103516e-05

需要指出的是，R 语言执行 for while 循环的效率比较低，因此不建议大量使用循环语句，推荐使用向量化计算方法，即将循环对象作为一个向量进行整体计算。 如果一定要对每一个值单独进行处理，在 R 中也有更高效的方法。

2.1.4 增加

对数据增加一列，可以使用 $ 运算符或者 [[]] 运算符，只要这一列的名字没有在数据中出现，直接赋值即可。新增的列出现在最后面。例如

demo.suspect0 <- demo.suspect # 备份一下
demo.suspect$married <- c(T, T, F)
demo.suspect
  name age gender height weight shoe.size married
1 张三  20      M   1.78     80        43    TRUE
2 李四  26      F   1.65     45        38    TRUE
3 王五  30      M   1.70     60        41   FALSE
demo.suspect[["children.count"]] <- c(2, 1, 0)
demo.suspect
  name age gender height weight shoe.size married children.count
1 张三  20      M   1.78     80        43    TRUE              2
2 李四  26      F   1.65     45        38    TRUE              1
3 王五  30      M   1.70     60        41   FALSE              0

我们也可以将一些计算值加入到列中，例如计算 BMI 指数

demo.suspect$BMI <- demo.suspect$weight / demo.suspect$height^2
demo.suspect
  name age gender height weight shoe.size married children.count      BMI
1 张三  20      M   1.78     80        43    TRUE              2 25.24934
2 李四  26      F   1.65     45        38    TRUE              1 16.52893
3 王五  30      M   1.70     60        41   FALSE              0 20.76125

以上这些操作，都可以使用一个函数 transform() 完成，该函数的用法类似于 select filter 两个函数，在后面可以直接指定新列的值

transform(demo.suspect0, BMI = weight / height^2, married = c(T, T, F), children.count = c(2, 1, 0))
  name age gender height weight shoe.size      BMI married children.count
1 张三  20      M   1.78     80        43 25.24934    TRUE              2
2 李四  26      F   1.65     45        38 16.52893    TRUE              1
3 王五  30      M   1.70     60        41 20.76125   FALSE              0

增加列的操作是比较简单的。对于增加行的操作，由于不同列的数据类型不同，因此比较复杂。

如果有行名，可以使用 [行名,列名] 的方式，指定一个新行名的索引，填入数据即可。 建议分别赋值，可保证列不改变。

row.names(demo.suspect0) <- demo.suspect0$name
demo.suspect0["赵六", "name"] <- "赵六"
demo.suspect0["赵六", "age"] <- 29
demo.suspect0
     name age gender height weight shoe.size
张三 张三  20      M   1.78     80        43
李四 李四  26      F   1.65     45        38
王五 王五  30      M   1.70     60        41
赵六 赵六  29   <NA>     NA     NA        NA

如果没有行名，可以使用函数 nrow() 计算以下当前的行数 \(r\)，使用 [r+1,列名] 的方式填入数据。

还可以使用 rbind() 函数增加一行，使用方法如下

demo.suspect <- rbind(demo.suspect, data.frame(
    name = "赵六",
    age = 19,
    gender = "F",
    height = 1.6,
    weight = 43,
    shoe.size = 37,
    BMI = 43 / 1.6^2,
    married = F,
    children.count = 0
))
demo.suspect
  name age gender height weight shoe.size married children.count      BMI
1 张三  20      M   1.78     80        43    TRUE              2 25.24934
2 李四  26      F   1.65     45        38    TRUE              1 16.52893
3 王五  30      M   1.70     60        41   FALSE              0 20.76125
4 赵六  19      F   1.60     43        37   FALSE              0 16.79687

但该函数效率较低，应尽量避免使用。同样地，R 中也有一个 cbind 函数，其用法和 rbind 类似，只是用于增加列，但该函数效率也比较低，因此不推荐使用。

2.1.5 修改

该部分所说的改，包括对名、类型、值的修改。

修改表数据列名和行名的方法和矩阵是类似的，都是使用 row.names() 和 col.names() 两个函数进行的。值得注意的是，不一定一次要全部修改。例如

colnames(demo.suspect)[7] <- "is.married"
demo.suspect
  name age gender height weight shoe.size is.married children.count      BMI
1 张三  20      M   1.78     80        43       TRUE              2 25.24934
2 李四  26      F   1.65     45        38       TRUE              1 16.52893
3 王五  30      M   1.70     60        41      FALSE              0 20.76125
4 赵六  19      F   1.60     43        37      FALSE              0 16.79687

修改类型，特指修改列的类型，方法还是使用 as 族函数，通过对一列进行整体操作，例如

demo.suspect$age <- as.integer(demo.suspect$age)
demo.suspect$children.count <- as.integer(demo.suspect$children.count)
demo.suspect
  name age gender height weight shoe.size is.married children.count      BMI
1 张三  20      M   1.78     80        43       TRUE              2 25.24934
2 李四  26      F   1.65     45        38       TRUE              1 16.52893
3 王五  30      M   1.70     60        41      FALSE              0 20.76125
4 赵六  19      F   1.60     43        37      FALSE              0 16.79687

修改值的方法和修改类型的方法基本一样，例如如果给 age 整体增 1，可以使用以下方式

demo.suspect$age <- demo.suspect$age + 1
demo.suspect
  name age gender height weight shoe.size is.married children.count      BMI
1 张三  21      M   1.78     80        43       TRUE              2 25.24934
2 李四  27      F   1.65     45        38       TRUE              1 16.52893
3 王五  31      M   1.70     60        41      FALSE              0 20.76125
4 赵六  20      F   1.60     43        37      FALSE              0 16.79687

当然也可以对某一个值单独进行赋值，例如将 王五 的 is.married 列的值修改为 T

demo.suspect[demo.suspect$name == "王五", "is.married"] <- T
demo.suspect
  name age gender height weight shoe.size is.married children.count      BMI
1 张三  21      M   1.78     80        43       TRUE              2 25.24934
2 李四  27      F   1.65     45        38       TRUE              1 16.52893
3 王五  31      M   1.70     60        41       TRUE              0 20.76125
4 赵六  20      F   1.60     43        37      FALSE              0 16.79687

与增操作类似，一般不对行进行整体的改操作。

2.1.5.1 删

删除一行或者一列的方法，等价于选择另外一些行或者列。例如如果要删除 children.count 这一列，或者删除 children.count 这一列的值小于 1 的，只需要选择其他列，即可。 而如果列比较多，要删的列只有一两个，进行查询操作并不是很方便。此时可以使用负整数索引进行删除。例如

demo.suspect[-c(4),-c(8)]

  • 用 c 函数创建一组键值对，键表示第一张表的连接字段，值表示第二张表的连接字段。

2.2 向量化计算

之前提到，R 语言中执行循环的效率是比较低的，提高效率的方法就是向量化计算，即以向量为单位进行计算。 如果向量之间各元素的操作相同，且比较简单，那么可以直接使用向量运算来进行，这是最为推荐的方式。 但是如果向量计算的操作比较复杂，这时就需要用到 apply 族函数了。这些函数主要包括以下几个：

  • apply 针对矩阵进行计算
  • lapply 针对列表进行计算
  • sapply 针对列表进行计算，结果会进行自动化简
  • vapply 针对向量进行计算，
  • …

例如，我们现在有一个 \(n \times n\) 的距离矩阵（这里使用 dist 函数生成，也可以从外部导入），我们需要找到每一列中排名第 20 的距离值。就可以使用 apply 函数进行计算。

2.3 空间数据操作

对于空间数据的操作，可以分成两类：

  1. 对属性进行操作。这个操作方法和之前所讲的对表数据的操作方法相同，只是操作的对象是 @data 插槽中的值。
  2. 对几何进行操作。这类操作有很多，相交、合并、投影……这里就不一一介绍。但这里主要讲一个操作——坐标系转换。
